Installation

Get started with Deca Framework in seconds. Just include the CSS and JavaScript files.

HTML 
<!-- CSS -->
<link rel="stylesheet" href="./style/styles.css">

<!-- JavaScript -->
<script src="./index.js"></script>
üí° Pro Tip

Place CSS in the <head> and JavaScript before closing </body> tag for optimal performance.

Customization

Easily customize the framework using CSS variables.

CSS 
:root {
  /* Primary Colors */
  --deca-primary: #6366f1;
  --deca-secondary: #8b5cf6;
  
  /* Semantic Colors */
  --deca-success: #10b981;
  --deca-danger: #ef4444;
}
