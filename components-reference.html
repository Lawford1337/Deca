Components Reference

Complete guide to all Deca Framework components

Buttons

Button Variants

Six color variants for different actions and contexts.

HTML

<button class="btn btn-primary">Primary</button>
<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-success">Success</button>
<button class="btn btn-danger">Danger</button>
<button class="btn btn-warning">Warning</button>
<button class="btn btn-info">Info</button>

Button Sizes

Three size options: small, medium (default), and large.

HTML

<button class="btn btn-primary btn-sm">Small</button>
<button class="btn btn-primary">Medium</button>
<button class="btn btn-primary btn-lg">Large</button>

Forms

Input Fields

Styled form inputs with focus states and validation support.

HTML

<form class="form">
  <div class="form-group">
    <label for="text-input">Text Input</label>
    <input type="text" class="form-control" placeholder="Enter text">
  </div>
  
  <div class="form-group">
    <label for="textarea">Textarea</label>
    <textarea class="form-control" rows="4"></textarea>
  </div>
  
  <button class="btn btn-primary">Submit</button>
</form>

Cards

Card Variants

Flexible card components for displaying content.

Image Card

Cards with images are perfect for visual content like blog posts or products.

Simple Card

Basic cards without images work great for text-based content and features.

New

Featured Card

Highlight special content with the featured card variant and badges.

HTML

<!-- Card with Image -->
<div class="card">
  <img src="image.jpg" class="card-image">
  <div class="card-content">
    <h4>Card Title</h4>
    <p>Card description</p>
    <button class="btn btn-primary">Action</button>
  </div>
</div>

<!-- Featured Card -->
<div class="card card-featured">
  <div class="card-content">
    <span class="badge badge-primary">New</span>
    <h4>Featured Card</h4>
  </div>
</div>

Alerts

Alert Types

Contextual alert messages for user feedback.

Success! Your action completed successfully.
Info: Here's some helpful information.
Warning! Please review this notice.
Error! Something went wrong.

HTML

<div class="alert alert-success">
                                    <strong>Success!</strong> Your action completed successfully.
                                    </div>

                                    <div class="alert alert-info">
                                    <strong>Info:</strong> Here's some helpful information.
                                    </div>

                                    <div class="alert alert-warning">
                                    <strong>Warning!</strong> Please review this notice.
                                    </div>

                                    <div class="alert alert-danger">
                                    <strong>Error!</strong> Something went wrong.
                                    </div>

Badges

Badge Variants

Small labels for highlighting or categorizing content.

Primary Secondary Success Danger Warning Info

HTML

<span class="badge badge-primary">Primary</span>
<span class="badge badge-success">Success</span>
<span class="badge badge-danger">Danger</span>
